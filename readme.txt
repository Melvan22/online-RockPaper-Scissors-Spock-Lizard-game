каталог server - это сервер игры, для его запуска необходимо перейти в каталог server и выполнить команду "node server.js" в командной строке
каталог game - это сама игра (основа создана с помощью react-create-app), для её запуска необходимо перейти в каталог game и выполнить команду "npm start" в командной строке


сервер использует порт 4001 на localhost

React приложение (игра) использует порт 3000 на localhost
